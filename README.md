# OpenClaw

OpenClaw is an AI-powered assistant platform.

## Features

- Multi-channel communication (currently supporting WhatsApp)
- Modern todo list application
- Extensible skill system
- Cloud-ready deployment

## Available Channels

- WhatsApp: Fully configured and operational
- Lark/Feishu: Integration available via lark-integration skill (setup required)

## Todo List Application

A modern, responsive todo list application is available in the `todo-app` directory, running on port 8081.

## Skills

Various skills are available to extend functionality, including the newly added Lark/Feishu integration.
